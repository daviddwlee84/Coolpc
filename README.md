# 庫PC 查價 KUPC

原價屋爬蟲配合 Sqlite3 資料庫桌面端程式

## 主題及資料庫路徑

``` bat
%AppData%\KUPC 
```

## 如何使用

```
解壓縮後
雙擊CopyThemeFileToAppdata.bat後
等待bat把ttk第三方主題(Sun-Valley-ttk-theme)移入使用者AppData中
其餘檔案就可以刪除,可以留下exe主程式就好了
```
